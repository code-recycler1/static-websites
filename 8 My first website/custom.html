Custom Styles

This page documents the design system and custom styles applied to the website.

Design System

Variables

The website uses CSS custom properties for consistent styling:

  • Colors: Primary, secondary, text, accent
  • Spacing: 8px base unit with sm(16px), md(24px), lg(32px)
  • Typography: System font stack with responsive base size
  • Effects: Transitions, shadows, and border radius

Color Scheme

The website uses a carefully selected color palette:

  • Primary: var(--primary-color) - Used for main elements
  • Secondary: var(--secondary-color) - Used for backgrounds
  • Text: var(--text-color) - Main text color
  • Accent: var(--accent-color) - Highlights and decorative elements

Dark mode automatically adjusts these colors for better visibility.

Layout Components

Grid System

Responsive grid layouts using:

  • Container with max-width: min(1200px, 95vw)
  • Grid gap: 24px (var(--spacing-md))
  • Flexible columns with minmax()

Components

  • Navigation: Sticky header with hover effects
  • Cards: Certification and skill cards with shadows
  • Timeline: Visual history for experience
  • Forms: Enhanced input styling and validation

Responsive Design

The website adapts to different screen sizes:

  • Mobile-first approach
  • Breakpoint at 768px for layout changes
  • Flexible grids and images
  • Collapsible navigation menu

Accessibility Features

  • Semantic HTML structure
  • ARIA attributes where needed
  • Focus states and keyboard navigation
  • Reduced motion preferences support
  • Color contrast compliance

Performance Optimizations

  • System font stack for better loading
  • Responsive images
  • Efficient CSS selectors
  • Print stylesheet support
